Mã cấu hình Javascript
Vui lòng nhập đầy đủ thông tin...

Hướng dẫn sử dụng:

  1. Nhập Tên đăng nhập và Mật khẩu ở trên.
  2. Nhấn nút "SAO CHÉP DÒNG NÀY" để copy mã được tạo.
  3. Mở tệp tin accounts.js trong thư mục dự án của bạn.
  4. Dán (Paste) dòng vừa copy vào bên trong khối window.ALLOWED_USERS = { ... }.
  5. Lưu tệp, chạy lệnh git push để cập nhật lên Vercel!
